# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices using Machine Learning.

A Linear Regression model is trained on the Housing dataset after performing preprocessing, feature engineering, training, and evaluation.

---

## 🚀 Features

- Data Cleaning
- Data Preprocessing
- Feature Engineering
- One-Hot Encoding
- Train/Test Split
- Linear Regression
- Model Evaluation
- House Price Prediction
- Model Saving (.pkl)
- Prediction CSV Export

---

## 📂 Dataset

Housing.csv

Target Variable:

- Price

Input Features:

- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status
